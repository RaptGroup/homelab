# CLAUDE.md

Guidance for Claude Code working in this repository.

## Repository Overview

The Rockingham Homelab repo. The lab is being rebuilt as a 6-node bare-metal
Kubernetes cluster running Talos Linux. This repository will hold the
Terraform that provisions the infrastructure and the Talos machine
configurations for the nodes.

The repo was reset from an earlier VM-based design (ArgoCD apps, Hypercore
VMs, custom Go services) and is intentionally empty for now. New structure
will be added as the rebuild progresses.
